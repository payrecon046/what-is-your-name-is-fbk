How to Implement on GitHub

1

Create a New Repository

Go to your GitHub account and create a new repository:

# Click the '+' in top-right and select 'New repository'
# Name it something like 'passive-income-guide'
# Make it PUBLIC (required for GitHub Pages)
# Check 'Add a README file'
2

Upload Your HTML File

Method A: Web Interface

  1. Go to your new repository
  2. Click 'Add file' → 'Upload files'
  3. Name it index.html
  4. Paste the code from window
  5. Commit changes

Method B: Git Commands

# Clone your repository
git clone https://github.com/yourusername/passive-income-guide.git

# Create index.html file
cd passive-income-guide
touch index.html

# Paste the code into the file and save
git add .
git commit -m "Add passive income guide"
git push
3

Enable GitHub Pages

Set up GitHub Pages to host your website:

  1. Go to your repository Settings
  2. Click 'Pages' in the left sidebar
  3. Under 'Source', select 'Deploy from a branch'
  4. Choose 'main' branch and '/ (root)' folder
  5. Click 'Save'

Your site will be available at:

https://yourusername.github.io/passive-income-guide

(May take up to 5 minutes to deploy)

4

Optional: Custom Domain

If you want a custom domain (like yourname.com):

1. Buy a domain

  • Namecheap ($5-10/year)
  • Google Domains
  • Cloudflare

2. Configure DNS

# Add CNAME file to repo containing your domain
echo "yourdomain.com" > CNAME
git add . && git commit -m "Add CNAME" && git push

# Set DNS records at your registrar

✅ Success!

You now have a fully hosted passive income guide website on GitHub Pages. You can:

  • Share the GitHub Pages link
  • Continue editing by pushing changes
  • Add more pages by creating additional HTML files
